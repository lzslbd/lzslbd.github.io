Java 中定义常量的几种方法

2017-03-11 Java 阅读量

主要有三种：接口、类和枚举。

接口：

  1. 在接口里定义域(接口中会对域自动加上 public static final ，使之成为常量)
  2. 类实现该接口即可在类中调用该常量

类：

直接用 public static final 定义常量。

枚举：

在枚举类中直接写上常量值。
